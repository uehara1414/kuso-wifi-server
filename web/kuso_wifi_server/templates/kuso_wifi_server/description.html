このサイトについて

Kuso Wifi Hub とは

以下のツイートから生まれた、某大学のWifiが繋がらないときに腹いせに押すボタン「kuso-wifi-button」のデータ集積所です。

学内のネットに繋がらなかったときに押すクソネットボタンが欲しい

— uehara (@NofaceYa) 2016年11月9日

「kuso-wifi-button」が押されたときのWifiのssid, ネットワークスピード, コメントを集め、そのデータをグラフ化します。

作者がちょうどDjangoの勉強をしていたので、練習として作成しました。

このサイトのリポシトリはこちら

kuso-wifi-button を入手したい

こちらからどうぞ。

Electron製で、Windows, OSX, Linux に対応しています。
